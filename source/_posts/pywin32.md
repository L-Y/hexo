## 用法 cmd下进入解压出来的目录，执行如下命令。

```bash


python pyinstaller.py [opts] yourprogram.py

主要选项包括：

    -F, –onefile 打包成一个exe文件。
    -D, –onedir 创建一个目录，包含exe文件，但会依赖很多文件（默认选项）。
    -c, –console, –nowindowed 使用控制台，无界面(默认)
    -w, –windowed, –noconsole 使用窗口，无控制台
```
